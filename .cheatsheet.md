# My VIM Cheat Sheet

# Custom Shortcut
    <C-P> : Show Buffer files
    <C-N> : Show files
    <C-F> : Show recent files
    gc : Comment out
    nt : NERDTree
    <Space>w: write
    <Space>. open .vimrc

    <leader>f : Goyo displays text centered
    <leader>o : checks orthography

# Default
    <C-z> : stop the job
        fg[0-9] : start the job
        jobs : show job list
